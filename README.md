# Online Shop

A full-stack e-commerce application built with React, Laravel, and MySQL.

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Vite

### Backend
- Laravel
- MySQL

## Getting Started

### Frontend Setup
```bash
# Navigate to frontend directory
cd frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

### Backend Setup
```bash
# Navigate to backend directory
cd backend

# Initial
echo "ทักเฟสหาพี่ฟีน"
```

## Features
- User authentication
- Product management
- Shopping cart
- Order management

## Project Structure
```
online-shop/
├── frontend/          # React frontend
└── backend/           # Laravel backend
```
